          Test Case 003 – User Menu

          Steps:

          1. Once on the home page, click on the menu button in top left to dropdown the menu.

          2. Test all the menu items:
            1st Menu Item should show in the logged in person’s name.
            “Home” should take the user to the home screen.
            “My Projects” should take user to the screen to select projects for which user is an IPM.
            “Help” should take user to the help screen.
            “Sign Out” should sign user out and when user logs back in, should ask for credentials.

           

          Test Case 004 –Search Project

          Steps:



          1. The following screen should be shown after clicking on “My Projects”:
          2. Enter partial or full Account Name, System Name, Modality, Project ID or SID in the search box. System should filter the projects shown.
          3. Select a sort type from the dropdown to sort projects by Account Name, Zone or Modality.

          Test Case 005 – See Project Details

          Steps:

          1. On the search project screen, once the desired project is searched for, click the project card for the project details
          2. IPM is shown the buttons for each section for project details. Click one of the buttons to view or edit details. “Back to Search” button takes the user back to the project search screen

          Test Case 006 –Project Detail Options

          Steps:

          1. On the project detail options screen, click one of the buttons to view or edit details. “Back to Search” button takes user back to the project search screen
          2. For example, to view/edit construction information, clicking the construction screen brings up details for the construction:

           

          Test Case 007 – Construction

          Steps:

          1. Construction screen is prefilled with the data available. Make appropriate changes to the data. Confidence level needs to be greater or equal to 60 to make a change to the site ready date. Some dates may appear disabled till their preceding question is set to “Yes”. For example, “Contractor Selected” needs to be “Yes” to enable the date for it.
          2. On most detail screens, scrolling all the way to the bottom shows buttons for saving, cancelling or navigating back to the project detail options screen
          3. Click the “Save” Button. A confirmation box should appear. Click “Ok”. System should save the changes and go to the project options screen. User may also see a warning message about Bill date being within 90 days.
          4. Click the “Cancel” Button. A confirmation box should appear. Click “Ok”. System should go to the home screen.

          5. Click the “Back” Button. A confirmation box should appear. Click “Ok”. System should go to the home screen.

           

           

          Test Case 008 –IPM Activity

          Steps:

          1. On the home screen, click the “IPM Activity” button.


          2. IPM’s may now add Activities in 1 of 3 ways. Type in your comments, click on the camera button, or upload a picture from your gallery. Once done, click on the save button.




          1. The IPM Activity history can be found just below the Save button.




           

          Test Case 009 –Site Plan Requests

          Steps:

          1. You may view Site Plan requests associated with your projects, by clicking on the Site Planning menu on the project details page.

           

           

           

          Test Case 010 –EMRF

          Steps:

          1. You may view EMRF’s associated with your projects, by clicking on the EMRF menu on the project details page.

          Test Case 011 –Contacts

          Steps:

          1. You may view contacts associated with your projects, by clicking on Contacts menu on the project details page. These include any contacts entered on the project site readiness section, as well as the AE, ASM, ZVP Service, and CE.
          2. You may initiate a call on your mobile device by touching any of the contacts’ phone numbers.

          Test Case 012 –Email Equipment

          Steps:

          1. You may view or email the equipment list associated with your projects, by clicking on “Email Equipment List” on the project details page.  Click on the + sign to expand for viewing or to include in email.
          2. Type in the recipients email address under “Email To:” then click on Email to send.
